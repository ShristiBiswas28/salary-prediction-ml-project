# Salary Prediction ML Project

## Overview
This project predicts Data Science job salaries using Machine Learning.

## Dataset
Kaggle Data Science Salaries dataset

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Random Forest Regressor
- Matplotlib

## Steps
- Data cleaning
- Feature encoding
- Model training
- Prediction
- Evaluation

## Results
- Model used: Random Forest
- Evaluation: MAE & R² Score
